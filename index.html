💡 ¿Cómo usar NutriAlfaro?
María Nuri, esta app es para ti. Aquí puedes ver todo lo que hay en la despensa, saber qué se está acabando y qué falta comprar.

🌡️ La barra verde arriba muestra qué tan completa está la despensa. Si está llena = verde, si falta mucho = roja.

🥘 Despensa: Toca el botón verde "➕ Agregar productos" para decir qué compraste hoy. También puedes tocar + o − en cada producto.

🛒 Faltantes: Cuando algo se acaba, aparece aquí automáticamente. También puedes tocar "📋 ¿Qué se acabó?" para marcarlo rápido.

👨‍🍳 Recetas: María, aquí verás platos que puedes hacer con lo que tienes. ¡Incluye recetas colombianas!

🔄 Reiniciar mes: Al final del mes, toca el botón naranja abajo para empezar de nuevo con mercado nuevo.
